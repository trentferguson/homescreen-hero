# Static Posters for Setup Wizard

This directory contains static poster images used as background animations during the setup wizard and login page.

## Image Requirements

- **Format**: JPG or PNG
- **Aspect Ratio**: 2:3 (portrait orientation, like a movie poster)
- **Recommended Size**: 500x750px minimum (larger is fine, will be scaled down)
- **Naming**: `poster-01.jpg`, `poster-02.jpg`, etc.
- **Quantity**: 20-40 images recommended for best visual effect

## Licensing

All images in this directory must be:
- Public domain
- Creative Commons licensed (with appropriate attribution)
- Your own original work
- Otherwise legally licensed for redistribution

**DO NOT** include:
- Copyrighted movie posters without permission
- Images from The Poster Database (TPDb)
- Any content that violates copyright

## Suggested Sources

- **Public Domain Films**: Archive.org, Wikimedia Commons
- **Creative Commons**: Flickr (with CC filter), Unsplash
- **NASA Images**: NASA Image and Video Library (public domain)
- **Custom Artwork**: Create your own film-themed designs

## Usage

Images placed in this directory will automatically be available to the application at `/posters/poster-01.jpg`, `/posters/poster-02.jpg`, etc.

The setup wizard will load these images and display them as an animated background grid.
